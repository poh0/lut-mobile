# HomeworkBox

Simple homework tracking app
- Using Room db
- [Link to showcase video](https://youtu.be/L7jk500Uwok) (Youtube)

## Run:

Clone this repository:
```sh
$ git clone https://github.com/poh0/lut-mobile.git
```
Assuming you have android studio installed, you can now open the project/HomeworkBox directory in Android studio and press the run-button.
